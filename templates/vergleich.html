Welches Modell ist das richtige für mich?

LEG - Lokale Elektrizitätsgemeinschaft

Wählen Sie eine LEG, wenn:

  • ✓ Sie eine grössere Nachbarschaft oder mehrere Gebäude zusammenschliessen möchten
  • ✓ Sie Zeit haben (Start ab 2026) und die neue Möglichkeit nutzen möchten
  • ✓ Sie eine hohe Autarkie vom Stromnetz und maximale PV-Nutzung anstreben
  • ✓ Sie bereits jetzt Nachbarn finden und sich organisieren möchten

Jetzt Nachbarn finden →

EVL/vEVL - Eigenverbrauchslösung

Wählen Sie eine EVL/vEVL, wenn:

  • ✓ Sie bereits heute starten möchten (nicht bis 2026 warten)
  • ✓ Sie eine einfache Lösung mit minimalem Aufwand suchen
  • ✓ Sie die Abrechnung komplett Ihrem Energieversorger überlassen möchten
  • ✓ Ihre Teilnehmer im gleichen Gebäude oder in direkter Nachbarschaft sind

Mehr erfahren →

ZEV/vZEV - Zusammenschluss zum Eigenverbrauch

Wählen Sie einen ZEV/vZEV, wenn:

  • ✓ Sie mehr Kontrolle über die Abrechnung wünschen
  • ✓ Sie spezielle Anforderungen haben, die EVL/vEVL nicht erfüllt
  • ✓ Sie bereits einen Abrechnungsdienstleister haben oder finden können
  • ✓ Sie den zusätzlichen Aufwand in Kauf nehmen möchten

Häufige Fragen

Kann ich von einer EVL/vEVL zu einer LEG wechseln?

Ja, wenn Sie aktuell eine EVL oder vEVL betreiben, können Sie ab 2026 zu einer LEG wechseln, falls Sie mehr Teilnehmer aus der Nachbarschaft einbinden möchten. Sprechen Sie mit Ihrem Energieversorger über die Details.

Welches Modell ist am günstigsten?

Das hängt von Ihrer Situation ab. EVL/vEVL haben meist niedrigere laufende Kosten, da Ihr Energieversorger die Abrechnung übernimmt. Bei ZEV/vZEV kommen Kosten für den Abrechnungsdienstleister hinzu. Die Kosten für LEG werden noch definiert.

Brauche ich für alle Modelle eine Solaranlage?

Nein, Sie können auch ohne eigene Solaranlage teilnehmen und Strom von Nachbarn beziehen. Mindestens ein Teilnehmer in der Gemeinschaft muss aber eine Produktionsanlage (meist Solaranlage) haben.

Was passiert, wenn die Sonne nicht scheint?

Bei allen Modellen beziehen Sie automatisch Reststrom aus dem Netz Ihres Energieversorgers, wenn Ihre lokale Produktion nicht ausreicht. Sie bezahlen dafür die normalen Strompreise.

Wie werde ich unabhängig vom Stromnetz?

Vollständige Unabhängigkeit ist kaum möglich und auch nicht das Ziel dieser Modelle. Aber Sie erhöhen Ihre Autarkie, indem Sie einen grossen Teil Ihres Stroms lokal produzieren und nutzen. Mit Batteriespeichern können Sie die Autarkie zusätzlich steigern.
