Kostenlos. Open Source. Für immer.

OpenLEG ist freie Infrastruktur für Schweizer Stromgemeinschaften. Keine versteckten Kosten, kein Vendor Lock-in, keine Daten die verkauft werden.

Für Bewohner

CHF 0

Kostenloser Einstieg vom Adresscheck bis zur vorbereiteten LEG-Anmeldung.

  • ✓ Adressprüfung und Matching mit interessierten Nachbarn
  • ✓ Strukturierter Gründungsablauf statt Einzelabsprachen
  • ✓ Dokumente für Vereinbarung, Verträge und Anmeldung
  • ✓ Transparenter Status bis zur Inbetriebnahme
  • ✓ Daten bleiben im LEG-Kontext, kein Datenverkauf

Wie wird das finanziert?

Infrastrukturkosten

Die Plattform läuft auf einem einzigen Schweizer Server für ca. CHF 500/Monat. PostgreSQL, Flask, Caddy. Keine teuren Cloud-Dienste.

Nachhaltiges Modell

Optionale professionelle Dienstleistungen (Onboarding-Support, SDAT-CH Integration, Schulungen) für Gemeinden und Netzbetreiber, die Unterstützung brauchen.

Selbst betreiben

OpenLEG ist Open Source (AGPL-3.0). Jede Gemeinde, jeder Kanton kann die Plattform selbst betreiben.

git clone https://github.com/wgusta/openleg.git
cd openleg
cp .env.example .env
docker compose up -d
