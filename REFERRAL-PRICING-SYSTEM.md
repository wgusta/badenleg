**Date**: November 16, 2025

## Core Principle: Three Paths to LEG

### Path 1: Do It Yourself (FREE)
**You can form and manage a LEG without BadenLEG**
- Research Swiss Mantelerlass regulations yourself
- Find neighbors manually (knock on doors)
- Draft contracts yourself (templates available online)
- Submit DSO notifications yourself
- Calculate billing yourself (spreadsheet)
- Manage community yourself

**Cost**: CHF 0
**Time**: 40-60 hours initial setup + 3-5 hours/month ongoing
**Complexity**: High (legal, technical, administrative)

**We openly provide**:
- Free guides: "How to Form a LEG Without BadenLEG"
- Template contracts (download free)
- DSO notification templates
- Billing calculator (Excel download)
- Step-by-step instructions

**Why we do this**:
- Transparency builds trust
- Empowers users
- Some people prefer DIY (that's okay!)
- Those who try DIY often come back (it's harder than it looks)

---

### Path 2: Use BadenLEG Service (PAID or FREE via Referrals)
**We handle everything for you**
- Automated neighbor matching
- Legal contracts (Swiss-compliant)
- DSO liaison and submission
- Automated billing (15-min intervals)
- Member dashboards + mobile app
- Ongoing support

**Cost**: See pricing below
**Time**: 2 minutes registration + 1 hour formation + 0 hours ongoing
**Complexity**: Zero (we handle it all)

**Pay with**:
- Money (standard pricing)
- Referrals (refer neighbors = free service)
- Hybrid (some referrals + reduced price)

---

### Path 3: Hybrid (Mix DIY + Service)
**Use BadenLEG for some parts, DIY others**
- Free matchmaking to find neighbors
- DIY formation (use our templates)
- Use our servicing platform (CHF 4.90/month)

**Cost**: Partial
**Time**: Moderate
**Complexity**: Medium

```python
# Database schema
class User:
    id = Integer(primary_key=True)
    email = String
    address = String
    referral_code = String(unique=True)  # e.g., "ANNA2025"
    referred_by = String  # referral code of referrer
    created_at = DateTime

class Referral:
    id = Integer(primary_key=True)
    referrer_id = Integer  # User who referred
    referee_id = Integer   # User who was referred
    created_at = DateTime
    status = Enum('registered', 'active_in_leg', 'churned')

class Discount:
    id = Integer(primary_key=True)
    user_id = Integer
    discount_type = Enum('formation', 'servicing')
    discount_amount = Decimal  # CHF or percentage
    valid_until = DateTime
    reason = String  # e.g., "3 active referrals"

# Calculate discounts
def calculate_user_discounts(user):
    # Count active referrals (users in LEG)
    active_referrals = Referral.query.filter_by(
        referrer_id=user.id,
        status='active_in_leg'
    ).count()

    # Count total referrals (registered)
    total_referrals = Referral.query.filter_by(
        referrer_id=user.id
    ).filter(Referral.status != 'churned').count()

    discounts = []

    # Formation discount (need 1 registered referral)
    if total_referrals >= 1:
        discounts.append({
            'type': 'formation',
            'amount': 199,  # 100% discount
            'description': 'Formation FREE (1 referral)'
        })

    # Servicing discount (need 3 active referrals)
    servicing_price = 14.90
    if active_referrals >= 3:
        servicing_price = 2.90
        description = '80% off servicing (3+ referrals)'
    elif active_referrals == 2:
        servicing_price = 4.90
        description = '67% off servicing (2 referrals)'
    elif active_referrals == 1:
        servicing_price = 9.90
        description = '33% off servicing (1 referral)'
    else:
        description = 'Standard servicing price'

    discounts.append({
        'type': 'servicing',
        'amount': servicing_price,
        'description': description
    })

    return discounts, active_referrals, total_referrals
```

```python
import secrets
import string

def generate_referral_code(user):
    # Create unique code: FirstName + random digits
    first_name = user.name.split()[0].upper()
    random_suffix = ''.join(secrets.choice(string.digits) for _ in range(4))

    code = f"{first_name}{random_suffix}"

    # Ensure uniqueness
    while User.query.filter_by(referral_code=code).first():
        random_suffix = ''.join(secrets.choice(string.digits) for _ in range(4))
        code = f"{first_name}{random_suffix}"

    user.referral_code = code
    user.save()

    return code
```

```python
    user.leg_status = 'active'

    referrals = Referral.query.filter_by(referee_id=user.id).all()
```
